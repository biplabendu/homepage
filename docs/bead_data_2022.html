Overview/Goals

Email from Deborah:

This data set comes from an experiment with harvester ants last summer. Harvester ant foragers bring in lots of seeds and parts of plants, some of which are rejected by ants inside the nest. Eventually, the refuse is carried out and put on the nest mound midden.

Here we presented small plastic beads coated in oil which the ants brought into the nest, and were later put back out on the midden, outside the colony, close to the nest entrance.

mod.validate <- function(mod) {
  which.mod <- mod
  ### Independence - 1
  p1 <- s.tidydat.mod %>% mutate(residuals = resid(which.mod), fitted_values = fitted(which.mod)) %>% 
    ggplot(aes(x=cum_prop, y=residuals)) +
    geom_abline(intercept = 0, slope = 0, col="red") +
    geom_jitter(width=0.01, shape=21, size=2) +
    theme_Publication()
    #### Note: if the points are NOT evenly scattered around the red line indicates that the model violates
    ####    the normality assumption (which is likely the case sense since cum_prop data are temporally correlated)
  ### Independence - 2
  p2 <- s.tidydat.mod %>% mutate(residuals = resid(which.mod), fitted_values = fitted(which.mod)) %>% 
    ggplot(aes(x=age_class, y=residuals)) +
    geom_boxplot() +
    geom_abline(intercept = 0, slope = 0, col="red") +
    theme_Publication()
    #### Note: most residuals for 5-9 year old colonies have positive values, suggesting that the covariate "age-class"
    ####    should be added to the model (as per Zurr's book, pg. 51)
  ### Independence - 3
  p3 <- s.tidydat.mod %>% mutate(residuals = resid(which.mod), fitted_values = fitted(which.mod)) %>% 
    ggplot(aes(x=age_obs, y=residuals)) +
    geom_abline(intercept = 0, slope = 0, col="red") +
    geom_jitter(width=0.01, shape=21, size=2) +
    theme_Publication()
    #### Note:
  ### Independence - 4
  p4 <- s.tidydat.mod %>% mutate(residuals = resid(which.mod), fitted_values = fitted(which.mod)) %>% 
    ggplot(aes(x=prop_foraging_30s, y=residuals)) +
    geom_abline(intercept = 0, slope = 0, col="red") +
    geom_jitter(width=0.01, shape=21, size=2) +
    theme_Publication()
  ### Independence - 5
  p5 <- s.tidydat.mod %>% mutate(residuals = resid(which.mod), fitted_values = fitted(which.mod)) %>% 
    ggplot(aes(x=bead_color, y=residuals)) +
    geom_boxplot() +
    geom_abline(intercept = 0, slope = 0, col="red") +
    theme_Publication()
  
  ## combine the plots and return as output
  return((p1+p2)/(p3+p4)/p5)
}

SO, where are we at?

There are issues we still need to deal with. (see Zurr 2013, section 2.5)

  • Independece issue 1: the residual v. cum_prop graph shows that the residuals are not randomly distributed around the 0,0 line; there is a trend which might be due to temporal correlation present in our data (cum_prop of one time point depends on the cum_prop of previous time point(s)).
  • Independece issue 2: the residual v. bead_color graph shows that there is “batch” effects; different replicate (bead_color used) shows slight different distribution of the residuals. We need to add bead_color as a random effect (see Zurr 2013 for ideas)
  • The model might improve by relaxing the assumption that the residual errors for different age-classes follow the same distribution (here, considered to be normally distributed around 0). As you can see from the histogram of residuals, the distribution is bell-shaped but not centered at 0.(See Zurr, section 2.8 for allowing heterogeniety)
